NvBlastExtSerializationSerializeFamilyIntoBuffer

Defined in extensions/serialization/NvBlastExtLlSerialization.h

uint64_t NvBlastExtSerializationSerializeFamilyIntoBuffer(void *&buffer, Nv::Blast::ExtSerialization &serialization, const NvBlastFamily *family)

Utility wrapper function to serialize an NvBlastFamily.

Allocates the buffer internally using the callack set in ExtSerialization::setBufferProvider.

Equivalent to: 

serialization.serializeIntoBuffer(buffer, family, Nv::Blast::LlObjectTypeID::Family);

Parameters

  • buffer – [out] Pointer to the buffer created.

  • serialization – [in] Serialization manager.

  • family – [in] Pointer to the NvBlastFamily to serialize.

Returns

the number of bytes serialized into the buffer (zero if unsuccessful).
