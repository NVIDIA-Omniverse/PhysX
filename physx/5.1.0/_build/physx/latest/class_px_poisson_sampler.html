class PxPoissonSampler : public PxUserAllocated

Sampler to generate Poisson Samples locally on a triangle mesh or a shape.

For every local addition of new samples, an individual sampling density can be used.

Subclassed by PxTriangleMeshPoissonSampler

Adds new Poisson Samples inside the sphere specified.

Parameters
