PxParticleClothBuffer#

Defined in include/PxParticleBuffer.h

class PxParticleClothBuffer : public PxParticleBuffer#

A particle buffer used to simulate particle cloth.

Deprecated:

Particle-cloth, -rigids, -attachments and -volumes have been deprecated.

See PxPhysics::createParticleClothBuffer.
