class PxArticulationSensor : public PxBase

A force sensor that can be attached to articulation links to measure spatial force.

See also

PxArticulationReducedCoordinate::createSensor

Public Functions

virtual void release() = 0

Releases the sensor.

Note

Releasing a sensor is not allowed while the articulation is in a scene. In order to release a sensor, remove and then re-add the articulation to the scene.

virtual PxSpatialForce getForces() const = 0

Returns the spatial force in the local frame of the sensor.

See also

setRelativePose, getRelativePose

Note

This call is not allowed while the simulation is running except in a split simulation during PxScene::collide() and up to PxScene::advance(), and in PxContactModifyCallback or in contact report callbacks.

Returns

The spatial force.

virtual PxTransform getRelativePose() const = 0

Returns the relative pose between this sensor and the body frame of the link that the sensor is attached to.

The link body frame is at the center of mass and aligned with the principal axes of inertia, see PxRigidBody::getCMassLocalPose.

See also

setRelativePose

Returns

The transform link body frame -> sensor frame.

virtual void setRelativePose(const PxTransform &pose) = 0

Sets the relative pose between this sensor and the body frame of the link that the sensor is attached to.

The link body frame is at the center of mass and aligned with the principal axes of inertia, see PxRigidBody::getCMassLocalPose.

See also

getRelativePose

Note

Setting the sensor relative pose is not allowed while the articulation is in a scene. In order to set the pose, remove and then re-add the articulation to the scene.

Parameters
