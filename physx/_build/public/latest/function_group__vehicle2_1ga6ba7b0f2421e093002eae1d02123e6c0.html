void PxVehiclePhysXRoadGeometryQueryUpdate(const PxVehicleWheelParams &wheelParams, const PxVehicleSuspensionParams &suspParams, const PxVehiclePhysXRoadGeometryQueryParams &roadGeomParams, const PxVehiclePhysXMaterialFrictionParams &materialFrictionParams, const PxReal wheelYawAngle, const PxVehicleRigidBodyState &rigidBodyState, const PxScene &scene, const PxConvexMesh *unitCylinderSweepMesh, const PxVehicleFrame &frame, PxVehicleRoadGeometryState &roadGeomState, PxVehiclePhysXRoadGeometryQueryState *physxRoadGeometryState)ÔÉÅ

Compute the plane of the road geometry under a wheel and the tire friction of the contact.

Note

PxVehicleRoadGeometryState::hitState will have value false in the event that the there is no reachable road geometry under the wheel and true if there is reachable road geometry under the wheel. Road geometry is considered reachable if the suspension can elongate from its reference pose far enough to place wheel on the ground.
