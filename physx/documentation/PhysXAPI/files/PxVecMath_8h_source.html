80 // Basic AoS types are
81 // FloatV - 16-byte aligned representation of float.
82 // Vec3V - 16-byte aligned representation of PxVec3 stored as (x y z 0).
83 // Vec4V - 16-byte aligned representation of vector of 4 floats stored as (x y z w).
84 // BoolV - 16-byte aligned representation of vector of 4 bools stored as (x y z w).
85 // VecU32V - 16-byte aligned representation of 4 unsigned ints stored as (x y z w).
86 // VecI32V - 16-byte aligned representation of 4 signed ints stored as (x y z w).
87 // Mat33V - 16-byte aligned representation of any 3x3 matrix.
88 // Mat34V - 16-byte aligned representation of transformation matrix (rotation in col1,col2,col3 and translation in
89 // col4).
90 // Mat44V - 16-byte aligned representation of any 4x4 matrix.
