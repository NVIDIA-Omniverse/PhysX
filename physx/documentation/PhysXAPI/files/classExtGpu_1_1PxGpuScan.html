inline

Performs an exclusive scan in place on the given array.

Parameters
[in,out]inAndOutBufGpu array with the integer data which gets transformed into its exclusive cumulative sum
[in]streamGpu stream on which the calculation is scheduled. To be sure that the scan finished, a synchronize call must be executed on that stream.
[in]numElementsToScanOptional: The number of elements that should get scanned. By default all elements are processed. The maximal number of elements is specified in the constructor.
