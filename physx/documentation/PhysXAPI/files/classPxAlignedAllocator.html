Detailed Description

template<uint32_t N, typename BaseAllocator = PxAllocator>
class PxAlignedAllocator< N, BaseAllocator >

Allocate aligned memory. Alignment must be a power of 2! – should be templated by a base allocator Allocator, which is used to access the global PxAllocatorCallback instance (used for dynamic data types template instantiation), which can align memory
