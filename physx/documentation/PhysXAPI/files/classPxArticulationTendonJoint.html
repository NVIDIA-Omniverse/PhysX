pure virtual

Sets the tendon joint coefficient.

Parameters
[in]axisThe degree of freedom that the tendon joint operates on (should correspond to a degree of freedom of the associated link's incoming articulation joint).
[in]coefficientThe scale that the axis' joint position is multiplied by when summing up the fixed tendon's length.
[in]recipCoefficientThe scale that the tendon's response is multiplied by when applying to this tendon joint.
Note
RecipCoefficient is commonly expected to be 1/coefficient, but it can be set to different values if needed.
