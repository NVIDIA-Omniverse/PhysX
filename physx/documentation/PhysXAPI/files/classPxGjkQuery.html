 Abstract interface for a user defined shape GJK mapping support. A user defined shape consists of a core shape and a margin. If the distance between two shapes' cores is equal to the sum of their margins, these shapes are considered touching. More...

Parameters
[in]aShape A support mapping
[in]bShape B support mapping
[in]poseAShape A transformation
[in]poseBShape B transformation
[in]contactDistanceThe distance at which proximity info begins to be computed between the shapes
[in]toleranceLengthThe toleranceLength. Used for scaling distance-based thresholds internally to produce appropriate results given simulations in different units
[out]pointAThe closest/deepest point on shape A surface
[out]pointBThe closest/deepest point on shape B surface

Returns
