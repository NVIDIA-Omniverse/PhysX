Detailed Description

Preprocessor to prepare particle cloths for simulation.

Preprocessing is done by calling partitionSprings() on an instance of this class. This will allocate the memory in the output object, partition the springs and fill all the members of the ouput object. The output can then be passed without any further modifications to PxUserParticleClothBuffer::setCloths().

See PxCreateParticleClothPreprocessor, PxParticleClothDesc, PxParticleClothOutput
