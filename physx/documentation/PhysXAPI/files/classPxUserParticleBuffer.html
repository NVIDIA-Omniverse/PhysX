Detailed Description

The shared base class for all particle user buffers, can be instantiated directly to specify granular and fluid particles.

See PxPhysics::createParticleBuffer.

A particle user buffer is a container that specifies per-particle attributes of a set of particles that will be used during the simulation of a particle system. It exposes direct access to the underlying GPU buffers and is independent of the scene and particle system. Particle user buffers can be added/removed from a particle system at any time between simulation steps, and transferred from one particle system to another.
