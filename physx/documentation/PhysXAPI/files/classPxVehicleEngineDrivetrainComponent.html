Detailed Description

Forward integrate the angular speed of each wheel and of the engine, accounting for the state of the clutch, gearbox and differential.

Provide vehicle data items for this component.

Parameters
[out]axleDescriptionidentifies the wheels on each axle.
[out]wheelParamsspecifies the radius of each wheel.
[out]engineParamsspecifies the engine's torque curve, idle revs and max revs.
[out]clutchParamsspecifies the maximum strength of the clutch.
[out]gearboxParamsspecifies the gear ratio of each gear.
[out]brakeResponseStatesstores the instantaneous brake brake torque to apply to each wheel.
[out]actuationStatesstores whether a brake and/or drive torque are to be applied to each wheel.
[out]tireForcesstores the lateral and longitudinal tire force that has developed on each tire.
[out]throttleResponseStatestores the response of the throttle to the input throttle command.
[out]clutchResponseStatestores the instantaneous clutch strength that arises from the input clutch command.
[out]differentialStatestores the instantaneous torque split between the wheels.
[out]constraintGroupStatestores the groups of wheels that are subject to constraints that require them to have the same angular or linear velocity.
[out]wheelRigidBody1dStatesstores the per wheel angular speed to be computed by the component.
[out]engineStatestores the engine rotation speed to be computed by the component.
[out]gearboxStatestores the state of the gearbox to be computed by the component.
[out]clutchStatestores the clutch slip to be computed by the component.
Note
If constraintGroupState is set to NULL it is assumed that there are no requirements for any wheels to have the same angular or linear velocity.
