Detailed Description

Forward integrate the momentum and pose of the vehicle's rigid body after applying forces and torques from the suspension, tires and anti-roll bars.

Retrieve pointers to the parameter and state data required to update the dynamic state of a rigid body.

Parameters
[out]axleDescriptionmust be returned as a non-null pointer to a single PxVehicleAxleDescription instance that describes the wheels and axles of the vehicle.
[out]rigidBodyParamsmust be returned as a non-null pointer to a single PxVehicleRigidBodyParams instance that describes the mass and moment of inertia of the rigid body.
[out]suspensionForcesmust be returned as a non-null pointer to an array of suspension forces and torques in the world frame. The suspension forces and torques will be applied to the rigid body to update *rigidBodyState*.
[out]tireForcesmust be returned as a non-null pointer to an array of tire forces and torques in the world frame. The tire forces and torques will be applied to the rigid body to update *rigidBodyState*.
[out]antiRollTorquemay be returned an optionally non-null pointer to a single PxVehicleAntiRollTorque instance that contains the accumulated anti-roll torque to apply to the rigid body.
[out]rigidBodyStateimust be returned as a non-null pointer to a single PxVehicleRigidBodyState instance that is to be forward integrated.
Note
The suspensionForces array must contain an entry for each wheel listed as an active wheel in axleDescription.
The tireForces array must contain an entry for each wheel listed as an active wheel in axleDescription.
If antiRollTorque is returned as a null pointer then zero anti-roll torque will be applied to the rigid body.
