Parameters
[out]axleDescriptionidentifies the wheels on each axle.
[out]transmissionCommandsspecifies the values of the thrust controllers that divert torque to the tank tracks.
[out]wheelParamsis an array describing the radius of each wheel.
[out]differentialParamsdescribes the operation of the tank differential by specifying the default torque split between all wheels connected to the differential and by specifying the wheels coupled to each tank track.
[out]differentialStatestores the instantaneous torque split between all wheels arising from the difference between the thrust controllers.
[out]constraintGroupStatestores the groups of wheels that are connected by sharing a tank track.
