)
inline

Add a wheel constraint group by specifying the number of wheels in the group, an array of wheel ids specifying each wheel in the group and a desired rotational speed relationship.

Parameters
[in]nbWheelsInGroupToAddis the number of wheels in the group to be added.
[in]wheelIdsInGroupToAddis an array of wheel ids specifying all the wheels in the group to be added.
[in]constraintMultipliersis an array of constraint multipliers describing the desired relationship of the wheel rotational speeds.
Note
constraintMultipliers[j] specifies the target rotational speed of the jth wheel in the constraint group as a multiplier of the rotational speed of the zeroth wheel in the group.

References PxVehicleLimits::eMAX_NB_WHEELS, and PX_ASSERT.
